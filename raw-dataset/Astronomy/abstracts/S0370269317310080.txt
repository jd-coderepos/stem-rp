We have systematically investigated the spin- 3 2 to spin- 1 2 doubly charmed baryon transition magnetic moments to the next-to-next-to-leading order in the heavy baryon chiral perturbation theory (HBChPT). Numerical results of transition magnetic moments and decay widths are presented to the next-to-leading order: μ Ξ c c ⁎ + + → Ξ c c + + = − 2.35 μ N , μ Ξ c c ⁎ + → Ξ c c + = 1.55 μ N , μ Ω c c ⁎ + → Ω c c + = 1.54 μ N , Γ Ξ c c ⁎ + + → Ξ c c + + = 22.0 keV, Γ Ξ c c ⁎ + → Ξ c c + = 9.57 keV, Γ Ω c c ⁎ + → Ω c c + = 9.45 keV.
